# GoSatck - Desafio: Conceitos do Node.js

Criamos um back-end em NodeJS para armazenar repositórios, com a possibilidade de dar likes nos repositórios.

## Instalação

```
yarn
```
ou
```
npm install
```

## Testes

```
yarn test
```
ou
```
npm run test
```
